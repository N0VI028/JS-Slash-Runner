# Disable incompatible options

Forcely disable/enable some settings in SillyTavern, which are not recommended to set to the opposite value during normal gameplay:

- Auto-fix Markdown → Off
- Trim Incomplete Sentences → Off
- Forbid External Media → Off
- Show &lt;tags&gt; in responses → Off
- Show {{char}}: in responses → On
- Show {{user}}: in responses → On
